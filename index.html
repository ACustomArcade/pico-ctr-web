Firmware Update

Flash new firmware via the PICOBOOT protocol over WebUSB. The device must be in BOOTSEL mode (hold BOOTSEL button while plugging in, or use the button above).

‚ö†Ô∏è Windows: PICOBOOT driver required

Windows requires the WinUSB driver for the PICOBOOT interface. Without it, the browser cannot communicate with the device for flashing.

  1. Download and run Zadig
  2. Put your device in BOOTSEL mode (hold BOOTSEL while plugging in)
  3. In Zadig, select Options ‚Üí List All Devices
  4. Select "RP2 Boot (Interface 1)" from the dropdown
  5. Set the driver to WinUSB and click Replace Driver
  6. Unplug and re-plug the device in BOOTSEL mode, then try connecting again

Alternative: You can also flash by dragging the .uf2 file onto the RPI-RP2 drive that appears in File Explorer when in BOOTSEL mode.
